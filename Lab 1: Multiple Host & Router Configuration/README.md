# Lab 1: Multiple Host & Router Configuration

## Objective
Configure three different subnets connected to a central Cisco router and verify communication between network devices.

## Network Topology

- Subnet 1: 172.20.15.0/28
  - Gateway: 172.20.15.1
  - 5 PCs

- Subnet 2: 172.20.15.16/28
  - Gateway: 172.20.15.17
  - 2 PCs
  - 2 Laptops

- Subnet 3: 172.20.15.32/28
  - Gateway: 172.20.15.33
  - 1 PC
  - 2 Servers
  - 1 Printer

## Devices Used

- Cisco 2911 Router
- Cisco 2960 Switches
- PCs
- Laptops
- Servers
- Printer

## Skills Demonstrated

- IPv4 Addressing
- Manual Subnetting
- Router Configuration
- Switch Configuration
- Default Gateway Configuration
- End Device Configuration
- Connectivity Testing
